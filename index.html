    Using the SC Filter Builder

    The following instructions offer basic details about how to use the SC filter builder tool. The basic workflow is straightforward:

    • Import some images to process - drag files onto the canvas, or load them via the Image management panel.
    • Click on an image thumbnail to see it in the preview panel.
    • Choose a starter filter, or import a previously downloaded filter.
    • Inspect how the filter affects your images. Use the minimap to pan and zoom into image detail.
    • Switch between basic (fast) and accurate preview modes depending on performance and accuracy needs.
    • Adjust the filter by changing values, or adding and removing filter actions.
    • Click a filter action in the graph (bottom of the page) to open its controls.
    • When satisfied, process and download the filtered images.
    • If you've tweaked the filter, you can download it for future use.

    Keyboard controls

    The web page supports the following (hopefully cross-browser standard) keyboard controls and shortcuts.

    Keyboard Navigation:

    • TAB to navigate forwards
    • SHIFT+TAB to navigate backwards
    • ENTER to select

    Additional input controls:

    • For "select" inputs – UP-ARROW, DOWN-ARROW
    • For "range" inputs – RIGHT-ARROW, LEFT-ARROW
    • For "color" inputs – UP-ARROW, RIGHT-ARROW, DOWN-ARROW, LEFT-ARROW
    • For checkbox toggling – SPACE
